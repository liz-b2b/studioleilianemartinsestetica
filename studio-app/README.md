# Studio Leiliane Martins — React + Vite + Tailwind

Este projeto é uma conversão do mockup estático para uma aplicação React moderna usando Vite e Tailwind CSS. Imagens são carregadas diretamente do Unsplash.

Como rodar (PowerShell):

```powershell
cd 'C:\Users\Alexandre\Documents\STUDIO LEILIANE MARTINS\studio-app'
npm install
npm run dev
# Abra http://localhost:5173
```

O projeto contém:
- `index.html` — entrada Vite
- `src/` — componentes React e estilos Tailwind
- `tailwind.config.cjs` e `postcss.config.cjs`

Observações:
- Para substituir imagens, edite `src/App.jsx` e troque as URLs do Unsplash por URLs próprias ou por imagens locais em `public/`.
- Se quiser que eu adicione integração com Bootstrap em paralelo, posso acrescentar componentes específicos ou migrar para uma stack que combine ambos.
