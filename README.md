# Studio Leiliane Martins Estética — Mockup estático

Arquivos incluídos:

- `index.html` — página principal com estrutura e placeholders.
- `css/styles.css` — estilos com paleta rosa pastel, tipografia e layout responsivo.

Como abrir:

1. Abra `index.html` no seu navegador (duplo clique ou `File -> Open`).
2. Para desenvolvimento, sirva a pasta com um servidor HTTP se desejar (ex.: `python -m http.server`).

Observações:

- As imagens são placeholders SVG/CSS para que você possa substituir facilmente por fotos de lifestyle e produto.
- Fontes do Google estão referenciadas no `index.html` — conexão com internet é necessária para carregá-las.
